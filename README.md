# Es Grades

## Setup

_If the `rails` command is not found, you can use the `./bin/rails` command instead._

1. Install dependencies

    ```bash
    bundle install
    ```

1. Create database

    ```bash
    rails db:migrate
    ```

1. Run the seeders

    ```bash
    rails db:seed
    ```

1. Start the server

    ```bash
    rails server
    ```
