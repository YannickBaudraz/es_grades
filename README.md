# Es Grades

## Setup

_If the `rails` command is not found, you can use the `./bin/rails` command instead._

1. Install dependencies

    ```bash
    bundle install
    ```

2. Create database

    ```bash
    rails db:migrate
    ```

3. Run the seeders

    ```bash
    rails db:seed
    ```

4. Start the server

    ```bash
    rails server
    ```
